## Завдання 2: Обчислення визначеного інтеграла методом Монте-Карло

### Опис роботи

- Обчислено визначений інтеграл функції `f(x) = x^2` на відрізку [0, 2] методом Монте-Карло.
- Для порівняння використано функцію `quad` з бібліотеки SciPy.

### Результати

```
Monte Carlo integral: 2.6705064512383307
Quad integral: 2.6666666666666665 Absolute error: 2.9605947323337504e-14
Difference: 0.0038397845716642287
```

### Висновкиautopep8 --in-place --aggressive --aggressive task_1.py task_2.py

Метод Монте-Карло дозволяє отримати наближене значення визначеного інтеграла. У даному прикладі результат, отриманий методом Монте-Карло, дуже близький до точного значення, обчисленого за допомогою функції `quad`. Абсолютна різниця між результатами становить приблизно 0.0038, що свідчить про достатню точність методу при великій кількості випадкових точок. Для задач, де аналітичне обчислення складне або неможливе, метод Монте-Карло є ефективним підходом.
